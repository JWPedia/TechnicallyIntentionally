About

Hello, World! I am Jeewoong "Skaliq" Chang, a computer science student in the University of Rochester.

This website was created as a personal project to learn the basics of front-end web development. This website is built to be minimal, fast, and responsive. It is written in plain html and hosted on Github Pages. No fancy styling, animations or plugins. Still, this being a technically minded website, Google's code-prettify and Mathjax are used.

I have a wide range of interests spanning computer science and technology, the biological sciences, media criticism, competitive Hearthstone, and deliberate lifestyle design. The archives page is a list of all of my articles and projects about these topics. Prospective employers should head over to the resume page instead.
